# Predictivo — Sitio + Diario

Este repo publica el sitio en GitHub Pages.

## Estructura
- `index.html` → Home
- `blog/` → Diario (lista de posts)
- `_posts/` → Entradas del diario (Markdown). Formato: `YYYY-MM-DD-titulo.md`

## Publicar un post
Crea un archivo en `_posts/`, haz commit, y aparecerá en `/blog/`.
